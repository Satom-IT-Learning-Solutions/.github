## 📌 C'est quoi cet espace ?

C'est ici qu'on centralise tout le travail pratique — labs, projets de cours, rendus, ressources partagées entre formateurs et élèves. Si t'as un doute sur où pusher ton projet ou comment utiliser l'org, t'es au bon endroit.

> **Règle d'or** : un repo par cours / projet / promo. On évite les fourre-tout.

## 📋 Conventions à respecter

Pour que tout reste lisible pour tout le monde :

**Nommer un repo**
```
[nom]-[sujet]
# Exemples :
thomas-iot-hackathon
sarah-vlan-lab
marc-ctf-writeup
```

**Structurer ses commits**
```
feat: ajout du dashboard Grafana zone A
fix: correction du flow Node-RED capteur température
docs: mise à jour README avec schéma réseau
lab: exercice VLAN trunk/access complété
```

**Branches**
- `main` → version stable / rendu final
- `dev` → travail en cours
- `feature/nom` → fonctionnalité spécifique
